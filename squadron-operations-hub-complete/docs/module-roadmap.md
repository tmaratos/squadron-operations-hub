# Module roadmap

## Phase 1: Operational core

- Authentication and setup
- Squadron profile
- Users, memberships, and duty positions
- Command dashboard
- Tasks and suspense tracking
- Audit log
- Functional areas
- Recurring requirements
- Basic documents
- Discord channel linking

## Phase 2: Continuity and administration

- Process library
- Meeting agendas and action items
- Approval workflows
- Staff handoff notes
- Reports
- Notification center
- Calendar integration
- Contact directory

## Phase 3: Functional-area depth

- Finance and funding pipeline
- Logistics and inventory
- Safety
- Aerospace Education
- Cadet Programs
- Emergency Services
- Communications
- Public Affairs
- Recruiting and Retention
- IT and systems

## Phase 4: Readiness and automation

- Self-inspection checklists
- Readiness scoring
- Corrective-action tracking
- Scheduled reports
- Escalation rules
- Automation recipes
- Dashboard customization

## Phase 5: Broader platform

- Multiple squadrons
- Group and Wing views
- Reusable process templates
- Mobile application
- Optional AI operational assistant
